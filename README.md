💳 Mini ATM System (JavaFX)

📌 Project Description

The Mini ATM System is a GUI-based application developed using JavaFX. This project simulates basic ATM functionalities such as user login, balance inquiry, cash withdrawal, deposit, and PIN change. It is designed to demonstrate core Java concepts along with GUI development.

🚀 Features:-
🔐 User Authentication (Login System)
💰 Balance Inquiry
💸 Cash Withdrawal
💵 Deposit Money
🔄 PIN Change Functionality
🧾 Transaction Handling
🎨 Styled User Interface using CSS

🛠️ Technologies Used:-
Java
JavaFX
CSS

📂 Project Structure:-

MiniATM/
│── Main.java
│── LoginPage.java
│── Dashboard.java
│── Transaction.java
│── ChangePIN.java
│── styles.css
│── README.md

▶️ How to Run:-

1. Clone the repository:
   git clone https://github.com/vaani004/MiniATM-

2. Open the project in any IDE (IntelliJ IDEA / VS Code / Eclipse)

3. Compile and run:
   javac Main.java
   java Main
